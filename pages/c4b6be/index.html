转换流IO流详解

# 简介

区分 Java 的输入和输出：把自己当成程序， 当你从外边读数据到自己这里就用输入（InputStream/Reader）， 向外边写数据就用输出(OutputStream/Writer)

Java 中将数据的输入输出抽象为流，流是一组有顺序的，单向的，有起点和终点的数据集合，就像水流。按照流中的最小数据单元又分为字节流和字符流。

  • **字节流：**以 8 位（即 1 byte，8 bit）作为一个数据单元，数据流中最小的数据单元是字节。
  • **字符流：**以 16 位（即 1 char，2 byte，16 bit）作为一个数据单元，数据流中最小的数据单元是字符， Java 中的字符是 Unicode 编码，一个字符占用两个字节。

# 获取编码
# getEncoding
public String getEncoding()
1

返回此流使用的字符编码的名称。

如果编码具有历史名称，则返回该名称; 否则返回编码的规范名称。

如果此实例是使用InputStreamReader(InputStream, String)构造函数创建的，则返回的名称对于编码而言是唯一的，可能与传递给构造函数的名称不同。 如果流已关闭，此方法将返回null 。

  • 结果

    该编码的历史名称，如果流已关闭， null

  • 另请参见：

    Charset

代码示例:

@Test
public void test0() {
    FileInputStream fis = null;
    InputStreamReader isr = null;
    try {
        fis = new FileInputStream("E:\\file\\IOTestFile.txt");
        isr = new InputStreamReader(fis, "UTF-8");
        log.info("此流使用的字符编码的名称:{}", isr.getEncoding());
    } catch (FileNotFoundException e) {
        e.printStackTrace();
    } catch (UnsupportedEncodingException e) {
        e.printStackTrace();
    }
}

# 读取
# read
public int read() throws IOException
1

读一个字符

  • 重写：

    read在类 Reader

  • 结果

    字符读取，如果已经达到流的结尾，则为-1

  • 异常

    IOException - 如果发生I / O错误

代码示例:

@Test
public void test0() {
    FileInputStream fis = null;
    InputStreamReader isr = null;
    try {
        fis = new FileInputStream("E:\\file\\IOTestFile.txt");
        isr = new InputStreamReader(fis, "UTF-8");
        int readNum = 0;

        do {
            //isr.read()是二进制 转成十进制的整数
            readNum = isr.read();
            if (-1 == readNum) {
                break;
            }
            log.info("readNum==>:{}", (char)readNum);
        } while (-1 != readNum);

    } catch (FileNotFoundException e) {
        e.printStackTrace();
    } catch (UnsupportedEncodingException e) {
        e.printStackTrace();
    } catch (IOException e) {
        e.printStackTrace();
    } finally {
        try {
            if (null != fis) {
                fis.close();
            }

            if (null != isr) {
                isr.close();
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

# read
ublic int read(char[] cbuf,int offset,int length) throws IOException
1
  • 注意: offset 从第几个字符开始读取 length: 最大读取长度 offset+ length不能大于 cbuf.length的长度 不能小于0

将字符读入数组的一部分。

  • Specified by:

    read在 Reader

  • 参数

    cbuf - 目的缓冲区

    offset - 开始存储字符的偏移量

    length - 要读取的最大字符数

  • 结果

    读取的字符数，如果已经达到流的结尾，则为-1

  • 异常

    IOException - 如果发生I / O错误

代码示例:

  @Test
    public void test0() {
        FileInputStream fis = null;
        InputStreamReader isr = null;
        char[] cbuf = new char[13];
        try {
            fis = new FileInputStream("E:\\file\\IOTestFile.txt");
            isr = new InputStreamReader(fis, "UTF-8");

            log.info("是否准备好被读取:{}", isr.ready());

            int readNum = 0;
            int offset = 2;
            int length = 13;

            do {
                //isr.read()是二进制 转成十进制的整数
                // 注意:
                // offset 从第几个字符开始读取
                // length: 最大读取长度
                // offset+ length不能大于 cbuf.length的长度 不能小于0

                if ((offset + length) >= 0 && (offset + length) <= cbuf.length) {
                    readNum = isr.read(cbuf, offset, length);
                    if (-1 == readNum) {
                        break;
                    }
                    log.info("readNum==>:{}", cbuf);
                } else {
                    log.info("读取长度大于cbuf的缓存长度");
                    break;
                }

            } while (-1 != readNum);

        } catch (FileNotFoundException e) {
            e.printStackTrace();
        } catch (UnsupportedEncodingException e) {
            e.printStackTrace();
        } catch (IOException e) {
            e.printStackTrace();
        } finally {
            try {
                if (null != fis) {
                    fis.close();
                }

                if (null != isr) {
                    isr.close();
                }
            } catch (IOException e) {
                e.printStackTrace();
            }
        }
    }

# 判断
# ready
public boolean ready() throws IOException
1

    ready在 Reader类

  • 结果

    如果下一个read（）保证不阻止输入，则为True，否则为false。 请注意，返回false并不能保证下一次读取将被阻止。

# 写入
# write
public void write(int c) throws IOException
1

flush()刷新流。 让数据快速写入 (注意:没有这个写入会报错)

写一个字符

  • 重写：

    write在类别 Writer

  • 参数

    c - int指定要写入的字符

  • 异常

    IOException - 如果发生I / O错误

代码示例:

@Test
public void test0() {
    OutputStream out = null;
    OutputStreamWriter osw = null;
    try {
        out = new FileOutputStream("E:\\file\\IOTestFile - 副本 (3).txt");
        osw = new OutputStreamWriter(out, Charset.forName("UTF-8"));
        log.info("获取编码:{}", osw.getEncoding());

        try {
            //写入数据
            osw.write(70);
            osw.write(71);
            osw.write(72);

            //刷新流。 让数据快速写入 (注意:没有这个写入会报错)
            osw.flush();
        } catch (IOException e) {
