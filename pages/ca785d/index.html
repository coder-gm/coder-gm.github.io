# 方法详细信息

# valueOf(char c)

public static Character valueOf(char c)
1

返回一个表示指定的char值的Character实例。 如果不需要新的Character实例，则该方法通常优先于构造函数Character(char)使用 ，因为该方法可能通过缓存经常请求的值而产生明显更好的空间和时间性能。 该方法将始终缓存范围为'\u0000'至'\u007F' （含）的值，并且可能将其他值缓存在此范围之外。

  • 参数

    c - 一个char值。

  • 结果

    一个 Character实例代表 c 。

  • 从以下版本开始：

    1.5
