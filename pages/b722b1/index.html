UUID 详解

# UUID 的概念

UUID（Universally Unique Identifier）：通用唯一识别码，是一种软件建构的标准。

UUID 目的是让分布式系统中的所有元素，都能有唯一的辨识信息，而不需要通过中央控制端来做辨识信息的指定。

UUID是指在一台机器上生成的数字，它保证对在同一时空中的所有机器都是唯一的。

# UUID的组成

UUID由以下几部分组合：

  1. 当前日期和时间，UUID的第一个部分与时间有关，如果你在生成一个UUID之后，过几秒又生成一个UUID，则第一个部分不同，其余相同。

  2. 时钟序列。

  3. 全局唯一的IEEE机器识别号，如果有网卡，从网卡MAC地址获得，没有网卡以其他方式获得。

UUID的唯一缺陷在于生成的结果串会比较长。

标准的UUID格式为：xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx (8-4-4-4-12)。

# 构造方法详细信息

# UUID(long mostSigBits,long leastSigBits)

public UUID(long mostSigBits,long leastSigBits)
1

构造一个新UUID使用指定的数据。 mostSigBits被用于最显著64位UUID和leastSigBits成为的至少显著64位UUID 。

  • 参数

    mostSigBits -的最显著位 UUID

    leastSigBits -的的最低显著位 UUID

# 方法详细信息

# randomUUID

public static UUID randomUUID()
1

静态工厂检索一个类型4（伪随机生成）的UUID。 UUID是使用加密强伪随机数生成器生成的。

  • 结果

    随机生成 UUID

# version

public int version()
1

与此UUID的版本号UUID 。版本号描述这怎么UUID产生。版本号具有以下含义：

  • 1个基于时间的UUID

  • 2 DCE安全UUID

  • 3基于名称的UUID

  • 4随机生成的UUID

  • 结果

    该版本号为 UUID

代码示例:

# variant

public int variant()
1

与此UUID的变式号。变体号描述的布局UUID 。变式号码具有以下含义：

  • 0保留用于NCS向后兼容

  • 2 IETF RFC 4122 (opens new window) （Leach-Salz），由这个类使用

  • 6保留，Microsoft Corporation向后兼容

  • 7保留供将来定义

  • 结果

    这个 UUID的变体号

代码示例:

# timestamp

public long timestamp()
1

与此UUID相关联的时间戳值。

60位时间戳值是由这个UUID的time_low，time_mid和time_hi字段UUID 。 所产生的时间戳记是从1582年10月15日UTC 15分钟以100纳秒为单位测量的。

时间戳值仅在具有版本类型为1的基于时间的UUID中有意义。如果此UUID不是基于时间的UUID，则此方法将引发UnsupportedOperationException。

  • 结果

    这个UUID的 UUID 。

  • 异常

    UnsupportedOperationException - 如果此UUID不是版本1 UUID

代码示例:

# clockSequence

public int clockSequence()
1

与该UUID相关联的时钟序列值。

14位时钟序列值由该UUID的时钟序列字段构成。 时钟序列字段用于保证基于时间的UUID中的时间唯一性。

clockSequence值仅在具有版本类型1的基于时间的UUID中有意义。如果此UUID不是基于时间的UUID，则此方法将抛出UnsupportedOperationException。

  • 结果

    这个 UUID的时钟顺序

  • 异常

    UnsupportedOperationException - 如果这个UUID不是版本1的UUID

代码示例:

# node

public long node()
1

与此UUID相关联的节点值。

48位节点值由该UUID的节点字段构成。 该字段旨在保存生成此UUID的机器的IEEE 802地址，以保证空间唯一性。

节点值仅在具有版本类型1的基于时间的UUID中有意义。如果此UUID不是基于时间的UUID，则此方法将抛出UnsupportedOperationException。

  • 结果

    该节点值为 UUID

  • 异常

    UnsupportedOperationException - 如果此UUID不是版本1 UUID

# compareTo

public int compareTo(UUID val)
1

将此UUID与指定的UUID进行比较。

如果UUID不同的最重要字段对于第一个UUID而言较大，则两个UUID中的第一个大于第二个UUID。

  • Specified by:

    compareTo在接口 Comparable<UUID>

  • 参数

    val - UUID要对这 UUID进行比较

  • 结果

    -1，0或1，因为这个 UUID小于，等于或大于 val
